#Options

-g
--glyph
Only print the glyph(s) for the file(s) passed as arguments (one-per-line), ex:
⚡ lsemoji -g / ~ # list gylphs for the filesystem root and your home folder
💻
🏡
-v
--version
Display version information; view the history.
-a
--all
Include hidden files (whose names begin with a ".") in file list
-c
--case
Enable case-sensitive name sorting
-f
--files
Only include files in the listing, excluding directories
-d
--dirs
Only include directories in the listing, excluding files
-k
Interleave files and directories (by default directories are listed first)
-l
--long
Long format, which displays permissions, owner & group, size, modification date, etc.
-e
--long
Similar to -l, but consecutive data will not be displayed, ex:
-m
Sort by file modification date (oldest first)
-r
Reverse sort direction
-s
--size
Sort by file size (smallest first); directories are sorted by the number of files within them
--prefix
Add the argument value to the beginning of each line of output in the listing, ex:
⚡ lsemoji --prefix="   " ~/files/ # prefix each line of output with 4 spaces
   📕  book.epub
   📅  calendar.ics
   💬  chat.ichat
   👤  contact.vcf
